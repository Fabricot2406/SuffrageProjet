Remplit la structure de données de classement avec les données du fichier CSV. 

Cette fonction prend une structure de données de type ballot et remplit sa liste de classement avec les informations provenant d\textquotesingle{}une matrice CSV représentant les préférences des votants. Chaque ligne de la matrice CSV représente les préférences d\textquotesingle{}un votant, et chaque colonne à partir de la 5ème contient le rang de préférence d\textquotesingle{}un candidat pour ce votant.
