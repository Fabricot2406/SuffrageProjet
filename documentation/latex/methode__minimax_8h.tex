Cette fonction implémente la méthode Minimax pour déterminer le gagnant d\textquotesingle{}une élection. Elle crée d\textquotesingle{}abord une matrice de duels à partir des bulletins de vote. Ensuite, elle initialise une liste d\textquotesingle{}arcs à partir de cette matrice. Elle alloue ensuite un tableau pour stocker le pire score de chaque candidat et initialise tous les scores à zéro. Elle utilise ensuite la fonction remplir\+Tab\+Pire\+Score pour remplir ce tableau avec les pires scores de chaque candidat. Enfin, elle détermine le candidat gagnant comme étant celui dont le pire score est le plus bas et affiche le nom du gagnant.
