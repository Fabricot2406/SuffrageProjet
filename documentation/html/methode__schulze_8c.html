Fonction permettant de déterminer si un candidat est vainqueur de condorcet. Processus : 1) On initialise une liste d'arcs à partir de la matrice de duels. 2) On trie la liste d'arcs dans l'ordre croissant. 3) On réduit la liste d'arcs jusqu'à obtenir un vainqueur.

Fonction permettant de déterminer si un candidat est vainqueur de condorcet. Processus : 1) On cherche pour chaque candidats si il est vainqueur, dans ce cas on retourne son indice. 2) Dans le cas contraire, on retourne -1.

Parameters

i : l'indice du candidat vainqueur si il y a un vainqueur.
