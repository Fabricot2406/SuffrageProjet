void afficher_classement ( List *  classement)

Affichage du classement des candidats.

Utilisation de la fonction list_reduce pour afficher les candidats dans l'ordre du classement. La fonction de réduction est une fonction qui affiche le candidat passé en paramètre et sa position dans le classement.

Précondition
La liste des arcs est non vide, triée par ordre décroissant de score.

classementLa liste des candidats du classement à afficher.

◆ est_vainqueur()

bool est_vainqueur ( List *  list_arc,
int  candidat 
)

Fonction permettant de déterminer si un candidat est vainqueur. Processus à suivre : 1) On parcourt la liste d'arc.

  • Si le candidat est impliqué dans un arc où il est perdant, alors il n'est pas vainqueur.
  • Si le candidat n'est impliqué dans aucun arc où il est perdant, alors il est vainqueur. 2) Si le candidat est impliqué dans aucun arc, alors il n'est pas vainqueur.
Paramètres
list_arcListe d'arc trié par ordre décroissant de différence de voix (score).
candidatCandidat dont on veut savoir si il est vainqueur.
Renvoie
true Si le candidat est vainqueur.
false Si le candidat n'est pas vainqueur.
