void calculer_uninominale_deux_tours ( ballot *  matrice,
FILE *  output 
)

Calcule le résultat de la méthode uninominale à deux tour. Processus : 1) On initialise un tableau de votes. 2) On cherche dans ce tableau les deux vainqueurs. 3) Si le score du vainqueur un est supérieur à 50%, alors on l'affique comme vainqueur, sinon on continue. 4) On cherche le vainqueur entre les deux (celui qui obtient le plus de voix en ne contant que leurs votes). 5) On affiche le vainqueur dépendant de qui a le plus haut score.

Parameters
matriceUne matrice de type ballot.
outputLe fichier de sortie dans lequel on écrit le résultat.
Precondition
La matrice ballot est initialisée et remplie.
Postcondition
Le vainqueur du scrutin est affiché.
