◆ verifierVote()

void verifierVote ( char *  fichier)

Vérifie la validité du vote en demandant à l'utilisateur d'entrer ses informations d'identification, de les hasher, puis de rechercher le hash correspondant dans un fichier CSV. La fonction effectue les étapes suivantes :

  1. Demande à l'utilisateur d'entrer son nom, prénom et clé d'identification.
  2. Hash les informations d'identification.
  3. Recherche le hash dans un fichier CSV contenant les informations de vote.
Parameters
fichierLe nom du fichier CSV contenant les informations de vote.
