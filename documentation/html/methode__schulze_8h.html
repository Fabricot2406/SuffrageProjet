void condorcet_schulze ( t_mat_int_dyn *  matrice_duel,
char **  candidats_nom,
FILE *  output 
)

Fonction permettant de déterminer si un candidat est vainqueur de condorcet. Processus : 1) On initialise une liste d'arcs à partir de la matrice de duels. 2) On trie la liste d'arcs dans l'ordre croissant. 3) On réduit la liste d'arcs jusqu'à obtenir un vainqueur.

Paramètres
matrice_duelMatrice de duel avec les résultats du vote.
candidats_nomListe des noms des candidats.
outputLe fichier de sortie dans lequel on écrit le résultat.
