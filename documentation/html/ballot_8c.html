ballot* classement_init ( ballot *  b,
t_mat_char_star_dyn *  classement_csv 
)

Remplit la structure de données de classement avec les données du fichier CSV.

Cette fonction prend une structure de données de type ballot et remplit sa liste de classement avec les informations provenant d'une matrice CSV représentant les préférences des votants. Chaque ligne de la matrice CSV représente les préférences d'un votant, et chaque colonne à partir de la 5ème contient le rang de préférence d'un candidat pour ce votant.

Parameters
bLa structure de données de type ballot à remplir.
classement_csvLa matrice CSV contenant les préférences des votants.
Returns
Un pointeur vers la structure de données ballot remplie.

◆ ens_pref_search()

int ens_pref_search ( List *  list,
int  rang_preference,
bool  find_index 
)

Recherche le rang de préférence dans une liste.

Cette fonction permet de rechercher le rang de préférence spécifié dans une liste de préférences. Elle peut être utilisée pour vérifier la présence du rang de préférence ou pour obtenir l'indice du premier élément correspondant.

Parameters
listLa liste de préférence dans laquelle effectuer la recherche.
rang_preferenceLe rang de préférence à rechercher.
find_index
Returns
Si find_index = true : indice du premier élément correspondant.
Si find_index = false : 1 si le rang de préférence est présent, sinon 0.
