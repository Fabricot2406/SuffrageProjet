◆ verif_chaine()

int verif_chaine ( char *  chaine,
int  prenom 
)

Fonction qui vérifie si la chaine est valide Processus pour chaque caractère de la chaine: CAS 1 : Lettre Prénom : La première lettre de chaque mot est en majuscule. nom : toutes les lettres sont converties en majuscules. CAS 2 : Espace ou tiret.

  • Il ne peut pas y avoir plus d'un espace ou d'un tiret consécutif
  • Il ne peut pas y avoir un ' ' ou '-' à la fin de la chaîne CAS 3 : Autre caractère
  • Erreur
    Paramètres
    chainela chaine à vérifier
    prenom1 si c'est un prénom, 0 si c'est un nom
    Renvoie
    int 1 si la chaine est invalide, 0 sinon
